Work together. ECMAScript 是 JavaScript 的标准。截至 2012 年，所有的现代浏览器都完整了支持 ECMAScript 5.1，旧式的浏览器至少支持 ECMAScript 3 标准。在2015年6月17日，ECMA国际组织发布了 ECMAScript 的第六版，该版本正式名称为ECMAScript 2015，但通常被称为 ECMAScript 6 或者ES6。自此，ECMAScript每年发布一次新标准。本文档目前覆盖了最新ECMAScript的草案，也就是ECMAScript2017。
